## Quick Reference

| Task | Command |
|------|---------|
| Install dependencies | `pnpm install` |
| Build all packages | `pnpm build` |
| Run tests | `pnpm test` |
| Validate (CI mode) | `pnpm agents validate` |
| Fix lint issues | `pnpm fix` |
